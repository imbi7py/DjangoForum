# DjangoForum

## QUICK START
```bash
# run django server
cd ~ && cd DjangoForum/forumapp 
python manage.py runserver

# migrate DB
cd ~ && cd DjangoForum/forumapp 
python manage.py migrate
python manage.py makemigrations polls

# run test 
cd ~ && cd DjangoForum/forumapp 
python manage.py test polls
```
